-- Instruction to use ABM validation templates

Step 1. Open source_EMME, source_Counts and source_Speed files
Step 2. Must click on "Enable Editing" on the top if it shows up
Step 3. Open any validation templates of your interest
Step 4. Must click on "Enable Editing" on the top if it shows up. But ignore "Enable Content" if there is any.
        All the values should be refreshed and referenced to source_EMME, source_Counts and source_Speed files
